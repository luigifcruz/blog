Projects

I’m currently focusing my free-time developing Open-Source software. The following are my main projects:

  • CyberRadio: An SDR based FM/AM radio for desktop accelerated with CUDA.
  • PiSDR Image: The SDR Linux Distro for the Raspberry Pi.
  • WeatherDump: Multi-platform toolkit for weather satellite reception.1
  • Kimera: Low-latency hardware accelerated codec based video streaming utility.

If you like my work and want to support my free open-source projects, you can donate or make a pull-request!
