Trying To Run DOOM With The Raspberry Pi 4 Vulkan Driver

2020-12-02 :: Luigi Cruz
#RaspberryPi  #Vulkan  #V3DV  #GZDOOM  #DOOM  #RBDOOM-3-BFG  #MESA 
Vulkan is the new kid in the block for GPU accelerated tasks for Computing and Graphics. This new standard was released by the Khronos Group in 2016. When compared to OpenGL, Vulkan gives the developer more control over the hardware. This approach is widely used by other standards like DirectX 12 and Metal. The Raspberry Pi Foundation is working with the Open-Source consultancy company called Igalia to develop Vulkan drivers for the Raspberry Pi 4 Broadcom SoC.
