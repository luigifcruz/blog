Making an Arduino Based Levitron

2016-12-12 :: Luigi Cruz
#Arduino  #PID  #Magnets 
Me and my college group made this machine for Physics Class, so I decided to show how we made it here. The final machine can levitate a magnet smoothly for days! What the hell is Levitron? In short, Levitron is an electromagnet controlled by a micro-controller that “levitates” a permanent rare-earth magnet by alternating the coil current. If done quickly, it creates an illusion of levitation. How it works?
